Explorando o Princípio da Indução Matemática

Uma Abordagem Visual com Faixas de Parábola

Prova do Passo de Indução

Área do K - 1 retângulos

Por hipotese de indução, para todo k ≥ 2, a expresão a seguir é válida:

A(k) = 1 k 3 · [ 1 2 + 2 2 + ... + ( k − 1 ) 2 ]

Manipulando essa expressão podemos reescrever a hipotese de indução do segunte modo:

k 3 · A(k) = [ 1 2 + 2 2 + ... + ( k − 1 ) 2 ]      (i)

Agora, usando a área A(k)recalculada dos (k - 1) retangulos inscritos na parábola, teremos:

A(k)recalculada = ( k k + 1 ) 3 · A(k)

O objetivo é provar que, se a nossa fórmula for válida para um caso genérico k (a hipótese de indução), ela também será válida para o caso seguinte, k + 1.

A expressão ao adicionar um novo retângulo


Essa etapa conecta a intuição visual do gráfico com a formalidade da expressão matemática. A equação principal deve demonstrar que A(k + 1) = A(k) + Ak + 1 (Área do Novo Retângulo) pode ser expandida para a expressão:

A(k + 1) = ( k k + 1 ) 3 · A(k) +   Ak + 1

A(k + 1) = ( k k + 1 ) 3 · A(k) + k 2 ( k + 1 ) 3

A(k + 1) = 1 ( k + 1 ) 3 · [ k 3 A(k) + k 2 ]

A(k + 1) = 1 ( k + 1 ) 3 · [ ( 1 2 + 2 2 + ... + ( k − 1 ) 2 ) + k 2 ]

Usando a hipotese de indução (i), temos:

A(k + 1) = 1 ( k + 1 ) 3 · [ 1 2 + 2 2 + ... + ( k − 1 ) 2 + k 2 ]

Ak + 1 = k 2 ( k + 1 ) 3

A(k + 1) = ( k k + 1 ) 3 · A(k) + 1 ( k + 1 ) · [ ( k + 1 ) − 1 k + 1 ] 2

Ak + 1 = k 2 ( k + 1 ) 3

Nessa fórmula, a primeira parte é associada a fórmula da nossa (a hipótese de indução).

E, a segunda parte é a fórmula para calcular a área do novo retângulo adicionado.

O Desafio Final: A Simplificação Algébrica

Como indicado pela última frase, "Agora, o desafio é simplificar essa expressão."

Este é o objetivo final de qualquer passo indutivo: trabalhar algebricamente a expressão combinada de A(k + 1) para provar que ela pode ser reescrita na mesma forma da fórmula original, mas com k + 1 no lugar de k.
