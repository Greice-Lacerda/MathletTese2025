A fórmula da nossa hipótese de indução para A(k) depende da soma das áreas dos k - 1 retângulos azuis (sem o degenerado) mais a área do novo retângulo laranja. Ou seja, a área total é:

Observe que a área gerada pelos (k - 1) retângulos é recalculada todas as vezes que adicionamos um novo retângulo. Assim, recalculando A(k), temos:

A(k)recalculada = A1 + A2 + A3 + A4 + A5 + A6 + ... + Ak - 1

A(k)recalculada =

( 1 k + 1 ) · ( 1 k + 1 ) 2 + ( 1 k + 1 ) · ( 2 k + 1 ) 2 + ... + ( 1 k + 1 ) · ( k - 1 k + 1 ) 2

A(k)recalculada = ( 1 k + 1 ) 3 · [ 1 2 + 2 2 + ... + ( k − 1 ) 2 ]

Agora, o desafio é calcular a área do novo retângulo.
