Instruções de Uso da Ferramenta Visual

Menu Principal

• Definição do PIM: Exibe a definição do PIM.

• O Problema: Exibe o problema proposto.

• Instruções: Exibe o guia de uso das ferramentas.



Menu Gerar Gráfico

• Número de partes: Digite um valor para n.

• Gerar Gráfico: Clique para desenhar o gráfico.

Menu Inserção


Depois de gerar o gráfico inicial, utilize estes botões para explorar o passo indutivo da demonstração:


• Adicionar Retângulo: Clique neste botão para adicionar os elementos visuais que compõem a soma ou a área relacionada à proposição para o números de partes (n) escolhido.


• Próximo Passo: Após construir a visualização para um determinado números de partes, este botão provavelmente premitirá avançar na demonstração.

Exemplo de Fluxo de Uso:

  1. Comece clicando em "O Problema" para entender a fórmula que será demonstrada.

  2. No campo "Número de partes", digite 2.

  3. Clique em "Gerar Gráfico" para ver a visualização do caso base (n = 2).

  4. Use os botões "Adicionar Retângulo" e "Próximo Passo" para ver a construção da prova.

  5. Para explorar mais, repita o processo inserindo valores maiores no campo "Número de partes" e observe como o padrão se comporta.
