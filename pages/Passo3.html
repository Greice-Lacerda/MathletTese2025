Prova de Indução para n = k+1

No Passo 2, mostramos que a fórmula da área é válida para alguns valores de `n`. Agora, vamos assumir que ela é verdadeira para **n = k**.

Área(k) = $\frac{(k-1)k(2k-1)}{6k^2}$

Adicionando o (k+1)-ésimo retângulo

A prova por indução exige que a fórmula funcione para $n=k+1$. Para isso, precisamos adicionar o próximo retângulo ao polígono.

A nova área total é a soma da Área(k) com a área do novo retângulo.

A Prova Algébrica: A nova Área Total

A nova área do polígono, com $k+1$ retângulos, pode ser expressa como:

Área(k+1) = Área(k) + Área do Retângulo (k+1)

Parabéns!

Você concluiu a prova do passo de indução. A fórmula é verdadeira para todos os números inteiros $n \ge 2$.
