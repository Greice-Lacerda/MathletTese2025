Uma Abordagem Visual com Faixas de Parábola
Passo 1: Caso Base

No Passo 1, verificamos que a fórmula é válida para o primeiro caso, n = 2 (Caso base).




A ( 2 ) = 1 2 · ( 1 2 ) 2 = 1 2 · 1 2 2 = 1 2 3 = 1 8 = 0,125
Passo 2: Hipótese de Indução

Assumimos que a fórmula é verdadeira para um número inteiro arbitrário, n = k (Hipótese de indução).


A ( k ) = 1 k 3 · [ 1 2 + 2 2 + ... + ( k − 1 ) 2 ]
Passo 3: Passo Indutivo

Agora, vamos para o passo de indução.

Clique no botão abaixo para explorar a provar da validade de:

A ( n ) = 1 n 3 · [ 1 2 + 2 2 + ... + ( n − 1 ) 2 ]

Prova de Indução para n = k + 1

O objetivo aqui é provar que para todo n natural maior ou igual a 2, a área A(n) do polígono retangular inscrito na parábola y = x2 no intervalo de [0, 1], pode ser expressa por:

A ( n ) = 1 n 3 · [ 1 2 + 2 2 + ... + ( n − 1 ) 2 ]

Pela Hipótese de Indução, assumimos que a fórmula da área A(k) funciona para n = k, do seguinte modo:

A ( k ) = 1 k 3 · [ 1 2 + 2 2 + ... + ( k − 1 ) 2 ]

Nosso desafio é provar que A(k) também funciona para n = k + 1. Antes de usarmos álgebra, vamos entender como isso funciona visualmente.

Construindo a Área(k + 1)

Abaixo está a representação da Área(k). Clique no botão para adicionar o retângulo que falta e formar a Área(k + 1).

Juntando as Peças

Perfeito! A nova área total é a soma da área original com o novo retângulo.

Área(k+1)   =   Área(k)   +   Novo Retângulo

Agora que entendemos o conceito visual, vamos traduzir isso para a matemática.

Traduzindo o Visual para a Matemática
