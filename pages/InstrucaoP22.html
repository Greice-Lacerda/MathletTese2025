A Hipótese de Indução - Etapa 1

Nesta fase, você investigará como a soma das áreas dos retângulos se comporta à medida que aumentamos o número de divisões (n) do intervalo.

O objetivo é observar um padrão nos casos específicos para, então, conseguir criar uma fórmula geral para um caso n = k qualquer.


Desafio 1: Caso n = 3

Sua primeira tarefa é analisar o cenário quando dividimos o intervalo [0, 1] em 3 partes iguais.

Utilize o gráfico para visualizar a base e a altura de cada retângulo e calcular a área do polígono retangular inscrito.

Desafio 2 e 3: Casos n = 4 e n = 5

O processo realizado no caso 1 deverá ser repetido para mais dois casos, aumentando o número de divisões.

Em cada um desses casos, você responderá às mesmas duas perguntas. O seu objetivo é anotar, calcular e comparar os resultados finais das áreas do polígono retangular inscrito.

Ao inserir a resposta correta em cada caso, uma tela de verificação mostrará o cálculo detalhado para que você possa comparar sua resposta.

Você consegue notar um padrão surgindo?

O Grande Salto: Generalizando para n = k

Este é o desafio mais importante e o coração do passo de indução!

Até agora, você calculou valores numéricos para as áreas. Agora, o objetivo é diferente: encontrar uma ou expressão algébrica que funcione para qualquer número k de divisões.


Como Encarar o Problema?

Olhe para o padrão, analise os resultados que você obteve para n = 3, 4, 5 e busque por uma fórmula da área total que se relacione com o valor de n.
