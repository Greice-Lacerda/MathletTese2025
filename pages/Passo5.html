Etiquetas Disponíveis

Arraste estas etapas para a área de ordenação.

Assumimos que A(k) é válida para n = k, temos: A ( k ) = 1 k 3 · [ 1 2 + 2 2 + ... + ( k − 1 ) 2 ]
Devemos provar que A(k + 1) é válida para todo k + 1.
Sabemos que a expressão da nova area será igual a:

A(k + 1) = A(k)recalculada + Ak+1 (Novo Retângulo)

Recalculando a area dos k - 1 retangulos inscritos para k + 1 partes, temos: A(k)recalculada = ( k k + 1 ) 3 · A(k)
A área do novo retangulo é igual a:

A ( k + 1 ) = k 2 ( k + 1 ) 3

Logo, a expressão da nova área será:

A(k + 1) = ( k k + 1 ) 3 · A(k) + k 2 ( k + 1 ) 3

Reorganizando a expressão da nova área :

A(k + 1) = 1 ( k + 1 ) 3 · [ k 3 A(k) + k 2 ]

Usando a Hipótese de Indução, temos:

A ( k + 1 ) = 1 ( k + 1 ) 3 · [ ( 1 2 + ... + ( k − 1 ) 2 ) + k 2 ]

Reorganizando a expressão obtida:

A ( k + 1 ) = 1 ( k + 1 ) 3 · [ 1 2 + ... + ( k − 1 ) 2 + k 2 ]

Deste modo, podemos concluir que A(k + 1) é válida para todo n = k + 1.

Resumo da Prova de Indução

1. Hipótese de Indução: Assumimos que A ( k ) = 1 k 3 · [ 1 2 + 2 2 + ... + ( k − 1 ) 2 ] é válida para n = k.
2. Objetivo (Tese): Devemos provar que A(k + 1) é válida para todo k + 1.
3. Montar A(k + 1): Sabemos que a expressão da nova area será igual a: A(k + 1) = A(k)recalculada + Ak+1 (Novo Retângulo)
4. Recalcular A(k): Recalculando a area dos k - 1 retangulos, temos: A(k)recalculada = ( k k + 1 ) 3 · A(k)
5. Área do Novo Retângulo: A área do novo retangulo é igual a A ( k + 1 ) = k 2 ( k + 1 ) 3
6. Expressão Completa: Logo, a expressão da nova área será: A(k + 1) = ( k k + 1 ) 3 · A(k) + k 2 ( k + 1 ) 3
7. Reorganizar: Reorganizando a expressão da nova área: A(k + 1) = 1 ( k + 1 ) 3 · [ k 3 A(k) + k 2 ]
8. Usar a Hipótese: Usando a Hipótese de Indução, temos: A ( k + 1 ) = 1 ( k + 1 ) 3 · [ ( 1 2 + ... + ( k − 1 ) 2 ) + k 2 ]
9. Reorganizar (Final): Reorganizando a expressão obtida: A ( k + 1 ) = 1 ( k + 1 ) 3 · [ 1 2 + ... + ( k − 1 ) 2 + k 2 ]
10. Concluir: Deste modo, podemos concluir que A(k + 1) é válida para todo n = k + 1.
